  Here's how to install the sbt eclipse plugin in sbt (step 2):
  mkdir -p ~/.sbt/plugins 
  echo 'addSbtPlugin("com.typesafe.sbteclipse" % "sbteclipse-plugin" % "2.1.2")' >> ~/.sbt/plugins/plugins.sbt
  sb
